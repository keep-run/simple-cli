# simple-cli
本仓库将从零开始，一步一步实现一个完整的前端脚手架。其中会涉及`babel`，`webpack`，`node`等知识。最终实现具有以下指令的cli工具：
- `simple-cli help`     ----帮助指令
- `simple-cli init`     ----初始化项目
- `simple-cli start`    ----启动项目
- `simple-cli build`    ----打包项目
